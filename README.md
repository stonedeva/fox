# Fox Compiler
This is the repository for the fox programming language

## Installation Guide
### Dependencies
- GCC / CC / CLANG
- NASM
- CMake

```console
git clone https://github.com/stonedeva/fox.git
mkdir -p build
chmod +x build.sh && ./build.sh
```
